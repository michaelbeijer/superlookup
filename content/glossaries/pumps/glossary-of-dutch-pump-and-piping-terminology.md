# Glossary of Dutch Pump and Piping Terminology (Process & Wastewater Engineering)

## Terms

| Dutch | English | Notes |
|------|---------|-------|
| pomp | pump | Generic |
| centrifugaalpomp | centrifugal pump | Most common pump type in wastewater installations |
| verdringerpomp | positive displacement pump | Generic class |
| schroefpomp | screw pump | Often used for sludge and viscous media |
| lobbenpomp | lobe pump | Hygienic applications |
| membraanpomp | diaphragm pump | Common for chemical dosing |
| doseerpomp | dosing pump | Polymers, coagulants, chemicals |
| circulatiepomp | circulation pump | Internal process loops |
| opvoerpomp | booster pump | Pressure increase |
| voedingspomp | feed pump | Upstream of process unit |
| terugspoelpomp | backwash pump | UF / membrane installations |
| recirculatiepomp | recirculation pump | Closed-loop operation |
| zuigleiding | suction line | Pump inlet |
| zuigzijde | suction side | Pump inlet side |
| persleiding | discharge line | **Preferred** over pressure line |
| perszijde | discharge side | Pump outlet side |
| retourleiding | return line | Recirculation |
| bypassleiding | bypass line | Flow diversion |
| verdeelleiding | distribution line | Feed distribution |
| persdruk | discharge pressure | Outlet pressure of pump |
| zuigdruk | suction pressure | Inlet pressure of pump |
| debiet | flow rate | Preferred over “flow” in patents |
| volumestroom | volumetric flow rate | More explicit |
| debietmeting | flow measurement | |
| debietmeter | flow meter | |
| elektromagnetische debietmeter | electromagnetic flow meter | Standard wastewater term |
| regelklep | control valve | Preferred patent term |
| gestuurde klep | control valve | Synonym of regelklep |
| afsluiter | shut-off valve | Isolation |
| terugslagklep | check valve | Non-return valve |
| overdrukventiel | pressure relief valve | Safety component |
| leiding | line | Use *line* rather than *pipe* internally |
| leidingwerk | piping | System as a whole |
