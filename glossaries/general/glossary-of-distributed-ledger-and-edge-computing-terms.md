# Glossary of Distributed Ledger and Edge Computing Terms (English-Dutch)

## Terms

| English | Dutch | Notes |
|--------|---------|-------|
| associated with | geassocieerd met / die verband houden met, "Geassocieerd met" is preferred for logical data relationships; "verband houden met" is standard legal phrasing. Avoid "gekoppeld aan" (reserved for physical coupling). |  |
| child distributed ledger | onderliggend gedistribueerd grootboek, In a hierarchy. Avoid "kind". |  |
| claim | conclusie, Strict terminology for patent law. Never use "claim" or "eis". |  |
| committed blocks | vastgelegde blokken, Blockchain context. Avoid "toegewijde" (devoted). |  |
| comprising | omvattende, Standard patent term implying an open set ("including at least"). Avoid "bestaande uit" (closed set). |  |
| data | data / gegevens, Treat as plural ("data die worden verwerkt"). "Gegevens" is often a safer alternative. |  |
| decode | decoderen |  |
| device | inrichting, Standard patent term for hardware/apparatus. |  |
| distributed ledger | gedistribueerd grootboek, Standardized term. Avoid "ledger" in Dutch text. |  |
| distributed ledger record | gedistribueerd grootboekrecord, Write as one compound word. |  |
| edge node | edge-knooppunt, Use hyphen for readability. Avoid "randknoop". |  |
| edge service instance | edge-service-instantie |  |
| indicator | indicator, "Aanduiding" is also acceptable, but "indicator" is common in IT/telemetry contexts. |  |
| method | werkwijze, Standard patent term. Avoid "methode". |  |
| orchestration system | orkestratiesysteem |  |
| parent distributed ledger | bovenliggend gedistribueerd grootboek, In a hierarchy. |  |
| parse | ontleden |  |
| processing circuitry | verwerkingsschakeling, Avoid "circuit" (physical loop); "schakeling" is the standard abstraction. |  |
| reinforcement learning | reinforcement learning, Usually left in English in Dutch IT patents. |  |
| reinforcement learning reward value | beloningswaarde voor reinforcement learning |  |
| retrieve | ophalen |  |
| schedule (verb) | inplannen, Avoid "schema" (noun). |  |
| state transition | toestandsovergang, "Toestand" is preferred over "status" for state machines. |  |
| subset | deelverzameling, Mathematical/Logic term. Preferred over "subset" in formal texts. |  |
| workload | werklast |  |
