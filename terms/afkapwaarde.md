# afkapwaarde

## Dutch

- **afkapwaarde** — Een bepaalde (arbitraire) drempelwaarde voor de uitslag van een test. Indien het testresultaat gelijk is aan of hoger is dan de afkapwaarde, wordt de test als positief beschouwd. [^1]
- afkappunt

## English

- cut-off value
- abandonment value, liquidation value *(Accounting)*
- forced-sale value *(sales)*

## Examples

| Dutch | English | Source |
|-------|---------|--------|
| De afkapwaarde voor een positieve test is 0,5. | The cut-off value for a positive test is 0.5. | — |
| Indien het testresultaat gelijk is aan of hoger is dan de afkapwaarde, wordt de test als positief beschouwd. | If the test result is equal to or higher than the cut-off value, the test is considered positive. | minerva-ebm.be |
